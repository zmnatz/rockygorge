# MDX Content Manager

This skill handles the creation and maintenance of content-rich pages using MDX.

## Guidelines
- **Location**: All content pages should be `.mdx` files located in the `pages/` directory.
- **Component Integration**: Use React components within MDX to add interactivity or complex UI elements. Ensure components are imported correctly at the top of the file.
- **Markdown Conventions**: Use standard GitHub-flavored markdown for text content.
- **Frontmatter**: Use frontmatter for page metadata (e.g., title, description, date) and ensure it aligns with the expected schema for content pages.
- **Consistency**: Maintain a consistent tone and structure across all `.mdx` pages.
